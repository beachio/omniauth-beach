# OmniAuth Nuwe

This gem contains the Nuwe strategy for OmniAuth.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-nuwe'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-nuwe
